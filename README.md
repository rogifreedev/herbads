# Herb Ads

Creative Intelligence Webapp fuer Meta Ads mit Next.js, Vercel, Supabase, Supabase Vector und AI Analyse.

## Start

Empfohlen: Node.js 20 verwenden.

```bash
npm install
npm run dev
```

- `IMPLEMENTATION_PLAN.md`: technische Umsetzung
- `DESIGN_SYSTEM.md`: Herb-Media UX, Farben, Fonts und Layout
